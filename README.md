# Zero Ticket

A team project for the **Agent X Hackathon**.

We work on a challenge from **HADAF** (Human Resources Development Fund) in the **Customer Service & Customer Experience** track.

## What we do

We look at support tickets to find **why customers contact**. Then we help reduce these repeat contacts.

## Run Locally

```bash
source .venv/bin/activate
streamlit run app.py
```
